# FreshCart - E-Commerce

A simple e-commerce web application built with **Next.js** and **Tailwind CSS**.

## Features

- **Authentication**: Login, Register, Forgot Password, Change Password
- **Homepage**: Hero section, Category slider, Featured products
- **Products**: Listing, Product details page
- **Categories**: Listing, Category detail pages
- **Brands**: Listing, Brand detail pages
- **Cart**: Add, Remove, Update quantity, Clear
- **Wishlist**: Add, Remove, Move to cart
- **Checkout**: Cash & Online payment
- **Orders**: Order history

## API

Uses the Route Academy E-Commerce API: `https://ecommerce.routemisr.com/api/v1`

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
